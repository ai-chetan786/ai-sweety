## ✨ Features

- 🗣️ **Multi-Model AI Chat** — Powered by Groq & OpenRouter for ultra-fast responses
- 🖼️ **AI Image Generation** — Create images from text prompts using HuggingFace
- ⚖️ **Legal & IT Problem Solver** — Get instant help with legal and tech questions
- 📰 **Live News Integration** — Real-time news powered by Tavily web search
- 👥 **User Authentication** — Secure login & registration via Supabase
- 🌍 **15 Languages Supported** — Hindi, Tamil, Telugu, Kannada, Malayalam, Marathi, Bengali, Gujarati, Punjabi, Urdu, Arabic, French, Spanish, Chinese & English
- 📱 **Fully Responsive** — Works beautifully on mobile and desktop
- ⚡ **Instant Deployment** — Hosted on Vercel with 38+ deployments

## ⚙️ Environment Variables

To run this project locally, create a `.env` file with:

```env
GROQ_KEY=your_groq_api_key
HUGGINGFACE_API_KEY=your_huggingface_key
OPENROUTER_API_KEY=your_openrouter_key
SUPABASE_PROJECT_URL=your_supabase_url
SUPABASE_ANON_KEY=your_supabase_anon_key
TAVILY_API_KEY=your_tavily_key
```

